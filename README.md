# Farm Project Website

A single-page responsive farm website built using **HTML** and **CSS only**.  
This project highlights farm operations, our story, latest news, and upcoming events — all in one clean layout.

---

## 🌱 Features
- **One-Page Design**: Smooth scrolling sections for easy navigation.
- **Farm Operations**: Overview of daily activities and services.
- **Our Story**: Background and mission of the farm.
- **Latest News**: Updates about farming practices and announcements.
- **Upcoming Events**: Showcase of farm-related events and gatherings.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile.

---

## 🛠️ Tech Stack
- **HTML5** for structure  
- **CSS3** for styling and responsiveness
